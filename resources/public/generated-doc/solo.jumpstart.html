solo.jumpstart

Namespace that will/can be loaded to jump-start server-side code.

See solo.jumpstart.servlet_container_initializer.

Note: This namespaces requires solo.core so that after being jump-started you may use solo.core via nREPL without the need to require/load solo.core explicitly.

jumpstart

(jumpstart)

Will/can be called from solo.jumpstart.servlet_container_initializer/-onStartup.

Can be over-written/redefined in runtime/production environments. This implementation just calls (solo.nrepl/start-server).
