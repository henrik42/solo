start-server

(start-server & {:keys [port host], :or {port 4005, host "0.0.0.0"}})

Starts a Swank server on given port/host. There can be atmost one running server at any time. A running server may be shut down by (stop-server). Throws exception if the server cannot be started. Returns the server.

Example:

(start-server :port 2001 :host "127.0.0.1")
