solo.jetty

Functions around Jetty.

-main

(-main & args)

Main entry point for starting a Jetty server.

Just calls (start-server) without any arguments. This function is just for testing (not a full CLI).

start-server

(start-server & {:keys [port host], :or {port 3000, host "0.0.0.0"}})

Starts a Jetty server on given port/host serving solo.web/app. Returns the server which can be shut down via (.close <server>). Throws exception if the server cannot be started.

Example:

(start-server :port 2001 :host "127.0.0.1")
