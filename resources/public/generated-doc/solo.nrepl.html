solo.nrepl

Functions around nREPL.

-main

(-main & args)

Main entry point for starting an nREPL server.

get-connection

(get-connection {:keys [host port]})

Returns a new connection to an nREPL server. If the connection cannot be established an exception is thrown.

remote-eval

(remote-eval nrepl-connection code)

Evaluates code (String) remotely via nrepl-connection. Returns value of evaluation. If eval on remote site throws exception, a RuntimeException will be thrown (remote exception type and stacktrace will be lost).

Note: code may have more than one form. code will be eval’ed but only the last value will be returned.

start-server

(start-server & {:keys [port host], :or {port 7888, host "0.0.0.0"}})

Starts an nREPL server on given port/host. Returns the server which can be shut down via (.close <server>). Throws exception if the server cannot be started.

Example:

(start-server :port 7888 :host "127.0.0.1")
